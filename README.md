# Журнал RDP подключений

Корпоративная система для учета и мониторинга RDP подключений с импортом логов, матрицей обновлений и разграничением прав доступа.

---

### Краткое описание
Веб-приложение для автоматизации ведения журнала RDP сессий, контроля выполнения обновлений и управления доступом к сервисам VSD. Система позволяет пользователям добавлять записи вручную или импортировать их из логов, а администраторам — управлять всеми аспектами работы.

### Ключевые возможности

| Категория | Функции |
|-----------|---------|
| **Учет RDP** | Ручное добавление, импорт логов, автоматическое определение ЛПУ по IP |
| **Управление** | Разграничение ролей, пагинация, фильтрация, экспорт в Excel |
| **Мониторинг** | Матрица статусов обновлений, статистика записей |
| **Интеграция** | API для всех операций, маппинг IP-ЛПУ, панель сервисов |
